# Expense Tracker

Simple expense tracker

## Basic requirements

- Wallet Balance
- Budgeting
- CRUD of expenses
- Time series list view of expenses
- Categories of different types of expenses

## Wishlist

- Monthly/weekly/bi-weekly expense tracker
- Graph view of expenses
